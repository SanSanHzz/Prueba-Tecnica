# Prueba-Tecnica

Este repositorio está preparado con estructura **MVC** para una prueba técnica. Permite gestionar usuarios, colaboradores, siembras y herramientas/maquinaria.

---

## 1️⃣ Nombre del Proyecto

# Proyecto de Gestión Agrícola

---

## 2️⃣ Descripción Breve

Aplicación RESTful para la creación, edición y consulta de usuarios, colaboradores, siembras y herramientas, con autorización basada en JWT.

---

## 3️⃣ Tecnologías Utilizadas

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- Bcrypt
- Multer (para subir archivos)
- Morgan
- CORS

---

## 4️⃣ Requisitos Previos

- Node.js >= v18
- MongoDB activo (local o en la nube)
- Variables de entorno (.env)

---
